[メガネ・サングラス通販の Oh My Glasses(オーマイグラスィズ)](http://www.ohmyglasses.jp/) を開発しているエンジニアが、日々の開発にまつわる情報を発信するための、エンジニアブログを開始します。

これを書いているのは、Oh My Glasses のエンジニアの白土 ([@kei\_s](https://twitter.com/kei_s)) です。

今回は第一回ということで、このブログをどうやって構築したか、ご紹介します。

## Oh My Glasses loves Ruby

Oh My Glasses の開発では、Ruby と Ruby on Rails を使っています。

## Oh My Glasses loves GitHub

コードの管理は GitHub を使っています。

## Octopress on GitHub Pages

そこで、このブログも、Ruby 製のブログシステムである [Octopress](http://octopress.org/) と、GitHub がWebページをホスティングする [GitHub Pages](http://pages.github.com/) を連携して使うことにしました。
